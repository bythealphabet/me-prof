Centered, glowing section title (Courier 700) with an optional monospace eyebrow and a sans subtitle — the recurring "What We Do" heading.

```jsx
<SectionHeading eyebrow="// What We Do" subtitle="Six ways we help Curaçao businesses win online.">
  Services built to convert
</SectionHeading>
```

`as` / `size` control the element and scale (`h1`–`h3`), `align` is `center` (default) or `left`, `glow` toggles the text-shadow.
