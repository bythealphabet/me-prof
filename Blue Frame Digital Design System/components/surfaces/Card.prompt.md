Base surface card — top-lit gradient, draft hatch texture, ambient navy shadow, no border lines.

```jsx
<Card padding="lg" hover>
  <h3>Fast, modern, built to last</h3>
  <p>Intentional engineering — not a slow site held together with plugins.</p>
</Card>
```

`padding` is `none` / `md` / `lg`; `hover` adds the lift + shimmer sweep; `as` swaps the element.
